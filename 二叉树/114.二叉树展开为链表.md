[114.二叉树展开为链表](https://leetcode-cn.com/problems/flatten-binary-tree-to-linked-list/)
```javascript
/**
 * @param {TreeNode} root
 * @return {void} Do not return anything, modify root in-place instead.
 */
// 分析题意：左子树全部为null
// 左子树合并右子树时，会合并在原本的右子树之前
// 因此需要先递归遍历整棵树，在后序遍历中操作
// 首先需要记录原本的左节点、右节点
// 将右节点更改为原本的左节点
// 这里需要判断一下，如果左节点（修改位置后的右节点）不存在时，就让根节点下的右节点
var flatten = function(root) {
    if (root == null) {
        return null
    }
    flatten(root.left)
    flatten(root.right)
    let left = root.left;
    let right = root.right;
    root.left = null;
    root.right = left;
    // 记录一下当前节点的链表
    let r = root;
    while(r.right !== null) {
        r = r.right
    }
    r.right = right;
    return root
};
```
